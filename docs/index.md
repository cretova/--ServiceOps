# 小微 ServiceOps 项目文档

这是一个面向面试展示的项目文档站，围绕“微信小微生态下，企业如何把小程序服务接入 AI 入口并完成质量治理”展开。

## 推荐阅读顺序

1. [项目选择逻辑](项目选择思考过程总结.md)
2. [能力可行性与价值指标](能力可行性与价值指标分析.md)
3. [MVP 前端说明](mvp/README.md)

## 项目一句话

> 企业或小程序服务商负责“把小微接进来”，小微 ServiceOps 负责“证明它接得对、接得稳、接得安全、接得有业务价值”。
